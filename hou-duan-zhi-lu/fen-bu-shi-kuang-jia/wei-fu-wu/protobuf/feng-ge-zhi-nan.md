# 风格指导

### 一.缩进

* 一行不要超过80个字符。
* 缩进使用两个空格。
* 字符串使用双引号。

### 二.文件结构

应该按照如下顺序：

1. License header (if applicable)
2. File overview
3. Syntax
4. Package
5. Imports (sorted)
6. File options
7. Everything else

### 三.大小写风格

* package名全小写。
* message名、enums名、service名、rpc方法名使用驼峰风格。
* message中field使用小写加下划线风格。
* enum的值使用全大写加下划线风格。
* repeated字段使用复数名称。
